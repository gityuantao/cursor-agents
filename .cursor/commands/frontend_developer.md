# 前端工程师命令

## 命令描述
启动前端工程师Subagent，负责用户界面的技术实现、前端架构设计、组件开发和性能优化。

## 使用方式
```
/frontend_developer
```

## 功能说明
- 选择合适的前端技术栈和工具
- 设计前端项目结构和组件架构
- 实现UI组件和业务组件
- 完成页面布局和交互逻辑
- 编写测试用例和性能优化
- 集成API接口和数据管理

## 输出文档
- FRONTEND_ARCHITECTURE.md - 前端架构文档
- COMPONENT_DOCS.md - 组件文档
- API_INTEGRATION.md - API集成文档
- DEPLOYMENT_GUIDE.md - 部署指南
- TESTING_GUIDE.md - 测试指南

## 适用场景
- 前端项目开发
- 界面技术实现
- 组件库开发
- 前端架构设计
- 性能优化和测试

## 协作流程
1. 接收设计规范和PRD文档
2. 进行技术选型和架构设计
3. 实现UI组件和页面功能
4. 集成后端API接口
5. 传递给DevOps工程师部署
